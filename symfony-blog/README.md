# Création d'un blog (CMS) avec Symfony 6, Bootstrap 5 et EasyAdmin 4
 Installation
--------------
Install the PHP dependencies and JS dependencies.
```sh
composer install
```
```sh
npm install
```
Installing assets
```sh
npm run dev
```
